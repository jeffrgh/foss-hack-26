# foss-hack-26
Project for FOSS hack 2026.


AI Page Builder for Frappe
One-line prompt → Frappe Builder page, generated by a local LLM.
Requirements

Python 3.10+
Ollama with llama3.1 pulled
Frappe with Frappe Builder running at localhost:8000

Setup
bashpip install -r requirements.txt
streamlit run app.py
Config
Edit core/frappe_api.py to change FRAPPE_URL, FRAPPE_USER, FRAPPE_PASS if needed.
